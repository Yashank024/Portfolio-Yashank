## 🚀 3D Implementation

### Interactive 3D Mech Model
This portfolio features a fully autonomous 3D mech robot with advanced behavior systems.

#### Technologies Used
- **Three.js r128** - WebGL 3D library
- **GLTFLoader** - 3D model loading
- **RGBELoader** - HDRI environment mapping

#### Model Specifications
- **File**: `mech_kaino.glb` (157MB high-detail model)
- **Format**: GLTF/GLB
- **Auto-scaling**: Automatically fits viewport
- **Position**: Right side of hero section

#### Professional Rendering
```javascript
// Advanced renderer settings
renderer.outputEncoding = THREE.sRGBEncoding;
renderer.toneMapping = THREE.ACESFilmicToneMapping;
renderer.toneMappingExposure = 1.2;
renderer.shadowMap.enabled = true;
```

#### Lighting System
- **Ambient Light** (1.0) - Base illumination
- **Hemisphere Light** (1.2) - Sky & ground bounce
- **Directional Light** (2.0) - Main sun with shadows
- **Fill Light** (0.8) - Shadow reduction
- **Blue Accent Light** (3.0) - Visual interest
- **HDRI Environment** - Realistic metallic reflections

#### Animation State Machine
The mech operates autonomously with a sophisticated behavior system:

**State: WALK**
- Walks forward 2 units (4 steps)
- Speed: 0.02 units/frame
- Smooth forward translation

**State: AIM_FIRE**
- 30° left-to-right sweep
- Auto-fires bullets every 10 frames
- Duration: 3 seconds (180 frames)
- Smooth rotation interpolation

**Bullet System**
- Orange glowing projectiles
- Speed: 0.25 units/frame
- Emissive material (glow effect)
- Auto-cleanup after 60 units

#### Material Optimization
```javascript
// Fixed for better visibility
material.metalness = 0.2;  // Reduced shine
material.roughness = 0.4;  // Smoother surface
```

### Performance
- **Lazy Loading**: Model loads asynchronously
- **Progress Tracking**: Real-time loading percentage
- **Error Handling**: Graceful fallbacks
- **Memory Management**: Automatic bullet cleanup
- **60 FPS**: Optimized animation loop

## 📦 Setup Instructions

### Prerequisites
None! Just a browser and the files.

### Installation

1. **Clone or download** this repository
2. **Add your photo**: Place `yashank.png` in `/assets/` folder (transparent background recommended)
3. **Open** `index.html` in a browser

### Recommended: Use Live Server

If using VS Code:
1. Install **Live Server** extension
2. Right-click `index.html`
3. Select "Open with Live Server"
4. Portfolio will open at `http://localhost:5500`

### File Structure
```
Portfolio/
├── index.html          # Main HTML file

├── script.js           # Animations & interactions
```
